Repository for the paper "Within-host viral growth and immune response rates predict FMDV transmission dynamics for African Buffalo"

Folders in this repository are: <br />
  1. Archive
  2. Finalized fitting
  3. Output analysis 

The <ins> Archive </ins> folder contains previous versions of the model fitting, including efforts to fit time series data, such as SAA data for innate immune response, which was not part of the finalized analysis.
